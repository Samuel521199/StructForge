## 技术要点

### 组件使用规范
- 所有页面必须使用 `@/components/common/base` 中的通用组件
- 禁止直接使用 Element Plus 组件（除了通用组件内部）
- 复杂业务组件放在 `@/components/common/business` 中

### 类型安全
- 所有 API 调用都有完整的类型定义
- 使用 TypeScript 严格模式
- 避免使用 `any` 类型

### 用户体验
- 加载状态提示（Loading 组件）
- 错误处理（统一的错误提示）
- 自动刷新（运行中的执行）
- 响应式设计
